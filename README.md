# LedososZZZ

Прошивка модуля мобильного робота для проекта:
**«Робот для обслуживания бортовых поверхностей корпусов кораблей Северного морского пути»**.

Репозиторий содержит код PlatformIO/Arduino для базового управления движением робота и приема команд по радиоканалу nRF24L01.

## Контекст проекта

Проект направлен на автоматизацию работ по обслуживанию бортовых поверхностей судов в условиях обледенения. Решение разрабатывается как роботизированный инструмент, который позволяет:
- снизить долю ручного труда;
- повысить безопасность работ;
- реализовать дистанционный запуск и контроль режимов движения/очистки.

## Что реализовано в этом репозитории

- Инициализация радио-модуля nRF24L01 (`uMQ` + `RF24`).
- Прием команд по адресу `ILUSH`.
- Управление двумя мотор-каналами (направление + PWM).
- Несколько режимов движения (как основа сценариев очистки):
  - `one` — чередование правого/левого привода;
  - `two` — прямолинейный ход;
  - `thr` — цикл вперед/назад;
  - иное значение — остановка.

## Аппаратная конфигурация

- Платформа: Arduino Nano (ATmega328P)
- Радиомодуль: nRF24L01 (`CE=D9`, `CSN=D10`)
- Моторы:
  - Логика мотора A: `D4`, `D8`
  - Логика мотора B: `D3`, `D2`
  - PWM: `D6` (A), `D5` (B)
- Объявлены входы линейных датчиков: `A0`, `A1`, `A2`, `A3`, `A6`

## ПО и сборка

Параметры из `platformio.ini`:
- `platform = atmelavr`
- `board = nanoatmega328new`
- `framework = arduino`
- `monitor_speed = 115200`

Зависимости:
- `nrf24/RF24`
- `joaoaugustocz/mpu6050_FastAngles`

## Команды PlatformIO

```bash
pio run
pio run -t upload
pio device monitor -b 115200
```

## Примечание

Этот репозиторий отражает программную часть управляющего модуля (движение и радиокоманды). Полный проект включает также механическую конструкцию, электронику, сценарии эксплуатации и интеграцию с верхним уровнем управления.
